Тема: Форматирование текста

Самостоятельное задание.

  1. Открыть папку, которую вы создали на первом уроке.
  2. Запустите программу Блокнот.(Можно использовать любой)
  3. Добавить тег p и в него вставить любой текст размером 500-1000 символов
  4. Стилизовать свой текст с помощью тегов для форматирование текста
  5. Сохраните файл
  6. Для просмотра Web-страницы используйте любой браузер (Google Chrome,Internet Explorer, Opera, Mozilla Firefox или другой). Для этого, не покидая программу Блокнот (сверните окно на панель задач), откройте личную папку и двойным кликом по файлу index.HTML откройте окно браузера.
