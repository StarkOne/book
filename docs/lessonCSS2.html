Тема: CSS ссылки.

Самостоятельное задание.

  1. Зайдите папку, куда вы сохраняете все файлы своего сайта.
  2. Откройте основной файл с HTML разметкой
  3. Создайтё ссылку
  4. Откройте файл style.css
  5. И примените каждый псевдокласс состояния ссылки к ней
  6. Сохраните файл
  7. Для просмотра Web-страницы используйте любой браузер (Google Chrome,Internet Explorer, Opera, Mozilla Firefox или другой). Для этого, не покидая программу Блокнот (сверните окно на панель задач), откройте личную папку и двойным кликом по файлу index.HTML откройте окно браузера.
